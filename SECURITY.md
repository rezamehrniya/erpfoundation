# Security Policy

## Reporting a vulnerability

Please report suspected vulnerabilities privately to repository maintainers.

Include:

- description of the issue
- potential impact
- reproduction steps
- suggested mitigation (if known)

Please avoid opening a public issue for sensitive vulnerabilities until maintainers acknowledge and coordinate disclosure.

## Scope notes

ERP Foundation is a public reference toolkit and does not include production secrets or deployment infrastructure. Security reports related to unsafe examples, validation bypasses, or data-leak patterns are in scope.
