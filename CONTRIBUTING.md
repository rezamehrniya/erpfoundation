# Contributing

Thank you for contributing to ERP Foundation.

## Scope

This repository accepts contributions related to:

- architecture documentation
- safe specifications
- examples
- validation and safety utilities

This repository does **not** accept private ERP Core implementation details.

## Development

```bash
pnpm install
pnpm lint
pnpm typecheck
pnpm test
pnpm build
```

## Pull requests

- Keep PRs focused and small.
- Add or update tests for behavior changes.
- Avoid adding customer-specific business logic.
- Do not include credentials or private internal data.
